# Dimple — Developer Portfolio

A modern, responsive, single-page personal portfolio website built with **HTML5, CSS3, and Vanilla JavaScript** — no frameworks, no dependencies beyond Google Fonts and Font Awesome icons.

## 🚀 Live Preview

Open `index.html` directly in any modern browser (Chrome, Firefox, Edge, Safari).

---

## 📁 Project Structure

```
portfolio/
│── index.html              ← Entry point
│
├── css/
│   ├── style.css           ← Main styles, CSS variables, layout
│   ├── animations.css      ← Keyframes, scroll reveal, micro-animations
│   └── responsive.css      ← Breakpoints for desktop/tablet/mobile
│
├── js/
│   ├── script.js           ← Navbar, scroll, hamburger, typing, form
│   └── animations.js       ← Scroll reveal observer, skill bars, particles

│   └── resume.pdf          ← Downloadable resume (add your own)
│
├── images/

│   └── icons/              ← Custom icons
│
└── README.md
```

## ✨ Features

- **Glassmorphism design** with dark `#0F172A` background
- **Particle canvas** in hero with connected dot network
- **Typing animation** cycling through roles
- **Scroll reveal animations** with staggered children
- **Animated skill progress bars** (triggered on scroll)
- **Sticky navbar** with glass blur on scroll
- **Mobile hamburger menu** fully accessible
- **Smooth scrolling** and active section highlighting
- **Responsive grid** — works on desktop, tablet, and mobile
- **Contact form** with simulated submission and toast notification
- **Back-to-top** button with smooth transition
- **Scroll progress bar** at the top of the page
- **SEO meta tags** and semantic HTML5

## 📝 Customization

1. **Personal info**: Edit `index.html` — update name, bio, links, projects
2. **Colors**: Edit CSS variables in `css/style.css` under `:root`
3. **Profile photo**: Replace `assets/profile.jpg`
4. **Resume**: Replace `assets/resume.pdf`
5. **Projects**: Update project cards in the `#projects` section
6. **Skills**: Adjust `data-width` values on `.skill-bar-fill` elements

---

## 📦 Sections
